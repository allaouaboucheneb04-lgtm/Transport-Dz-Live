Ajout activation/désactivation ligne:
- Bouton Désactiver/Réactiver sur chaque ligne admin.
- Désactiver une ligne met aussi tous ses arrêts active:false.
- Les lignes/arrêts désactivés ne s'affichent plus côté client.
- Recherche trajet ignore les lignes désactivées.
- Admin peut réactiver la ligne et ses arrêts.
