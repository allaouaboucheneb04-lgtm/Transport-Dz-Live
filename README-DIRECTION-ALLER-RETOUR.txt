Sens aller / retour ajouté:
- Chaque arrêt peut être: aller, retour, both.
- Le moteur trajet respecte le sens.
- Le chauffeur choisit son sens de travail.
- Les segments affichent le sens.
- Par défaut les arrêts importés sont both.
- Pour plus de précision: modifier les arrêts importants et mettre leur direction.
