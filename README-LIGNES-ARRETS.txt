Correction arrêts par ligne:
- Un arrêt doit avoir un lineId.
- La carte ne relie plus les arrêts de lignes différentes.
- Si une ligne est sélectionnée: seuls ses arrêts sont affichés et reliés.
- Si 'Toutes les lignes' est sélectionné: chaque ligne est tracée séparément.
- Un arrêt sans ligne ne peut plus être ajouté depuis le formulaire.
